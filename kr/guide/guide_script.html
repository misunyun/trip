기본 사항 - 구분

모든 기능의 코드는 위에서 정의되고, 하단에서 컨트롤(실행) 합니다.

기본 사항 - 컨트롤

대문자 로 시작하는( ex : SET_ ) 함수, 변수는 모든 기능에서 전역(스크립트 파일 내)으로사용할수 있습니다.

DOMreadyInit

jquery의 $(document).ready에 전달할 하나의 init함수 입니다

READY_ ...

위의 DOMreadyInit가 실행하는 함수들입니다. 순서대로 실행됩니다.

READY_beforeSet : 페이지에서 기본적으로 사용할 요소들을 추가하고, 정의합니다 (ex:최초에 dimd element를 body아래에 추가)

READY_customSet : 페이지에서 필요한 기능들 단위 리스트입니다.

READY_afterSet : 페이지 로드 마지막 단계 기능 리스트 입니다.

SET_ ...()

나누어진 기능들 단위입니다. 실행시 조건이 되면 , 페이지에서 기능이 부여됩니다. 대부분 로드시 실행(READY_customSet)되는 전제입니다

ACT_ ...()

모든 코드 내에서 공통으로 사용 가능한 기능입니다. (dimd가 필요한 상황에서는 ACT_dimdOn()을 실행합니다)

SUB_ ...(function)

어떤 기능에 함께 실행되도록 하는 함수를 추가합니다. (ex : SUB_dimdOff( 딤드off시 함께할 함수 ));

IS_ ...

전역으로 관리하는 상태 값입니다.
(ex : IS_dimdOn : 현재 dimd가 열려있으면 true, 닫혀있으면 false)

dimd 관련

div class="dimd_all"는 페이지 로드시 자동으로 body바로 아래에 붙여집니다.

ACT_dimdOn()

공통 dimd를 올립니다

dimd가 요구되는 동작에 함께 사용합니다. 스크롤을 막는 기능이 실행됩니다.

ACT_dimdOff()

dimd를 클릭하면 dimdOff를 기본적으로 실행합니다. 그 외의 딤드를 닫게 할 상황이 있을경우 사용합니다.

SUB_dimdOff( function )

dimd 닫기시에 콘솔을 출력하는 기능을 추가합니다.

인자로 받은 함수를 dimd 닫기 시 함께 실행하도록 추가합니다.

IS_dimdOn

현재 dimd가 on인지에 대한 상태값입니다. true/false
